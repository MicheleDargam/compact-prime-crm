# Roadmap do Projeto Compact Prime CRM

---

# FASE 0 — Estruturação

Objetivo:
Preparar ambiente e arquitetura do projeto.

## Etapas
- configurar Antigravity IDE
- configurar Claude Code
- organizar documentação
- criar estrutura de pastas
- configurar Docker
- configurar GitHub

Status:
Em andamento

---

# FASE 1 — MVP CRM

Objetivo:
Criar primeira versão funcional do CRM.

## Funcionalidades
- login
- dashboard
- CRM Kanban
- cadastro de leads
- pipeline comercial
- integração WhatsApp
- IA básica
- mobile responsivo

Status:
Planejamento

Futuro
